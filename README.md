# Phoenix Layout Calculator

A lettering layout tool for sign painters.

Calculates:

- Stem widths
- Counter widths
- Letter spacing
- Word spacing

using traditional Phoenix (Condensed Gothic) lettering rules.

## Current Version

Python prototype.

## Planned Features

- Web App (PWA)
- Offline support
- Mobile friendly interface
- Multiple themes
- Margin calculator
- Additional lettering styles

## Why?

Because sometimes you just want to paint the sign instead of doing fraction math.
